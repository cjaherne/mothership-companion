# UI Artwork Assets

Place the following images in this directory for the Artwork Assets Placement plan:

| File | Usage |
|------|-------|
| `header-bg.png` | Header background image, displayed from ~1/3 into the screen |
| `planet-map-placeholder.png` | Planet Map panel when in prologue (e.g., distant planet, orbital view) |
| `locations-placeholder.png` | Locations panel when in prologue (e.g., ship interior, industrial scene) |

Optional (for future phases):
- `background-panel.png` - Briefing Background panel
- `home-hero.png` - Home screen
- `character-panel.png` - CharacterList / NpcSelector panels
- `session-bg.png` - Voice session
- `sidebar-accent.png` - Sidebar
